# Thermal Voltage Fluctuations in Resistors

Experimental measurement and analysis of thermal voltage fluctuations in resistors (Johnson noise) using amplified signals and digital signal processing techniques.

## Overview

This project investigates the measurement of thermal voltage fluctuations generated by resistors due to the thermal motion of charge carriers (Johnson Noise). Due to the scale of fluctuations, the project involved signal amplification and analysis in order to mitigate the impact of other, uncorrelated noise sources and extract the signal of interest from background noise sources.

This involved the calibration and characterizatoin of the experimental measurement setup, including: characterization of the gain and frequency dependence of the amplifier response; and the characterization of frequency impact on the digital recording device (in this experiment, an external USB sound card and Audacity recording software). Signal processing techniques included Fourier analysis via power spectral density (PSD) and cros spectral density (CSD) analysis, and the use of cross-correlation to accurately and honestly approximate our experimental uncertainty.

The full experimental methodology, analysis, and results are documented in the accompanying paper:

johnson_noise_analysis_paper.pdf

## Methods

The experiment consisted of several key stages:
1. System Calibration - Characterization of the gain and frequency dependence of the stereo amplifier, characterization of the frequency dependence of the external sound card.
2. Data Acquisition - Thermal noise from resistors was amplified and recorded as .WAV files using a stereo amplifier, external USB sound card, and audio recording software.
3. Signal Processing - Thermal noise signals were analyzed using PSDs, CSDs, and experimental uncertainty was determined using cross-correlation
4. Data Analysis - A band of frequencies was decided upon through which the PSD/CSDs of all resistors behaved as expected and averaged PSD/CSDs were calculated across these bands for each resistance. From this a linear fit of average spectral density as a function of resistance was performed to determine an experimental value for Johnson Noise; this was compared against theoretical predictions.

## Requirements

The analysis notebook requires the following Python libraries:
- Numpy
- Matplotlib
- Scipy
- Warnings

## How to Run

Open johnson_noise_analysis.ipynb and run the cells to reproduce the signal processing analysis and figures used in the final report.

The notebook reads waveform data from the raw_data/ directory.

The LaTeX files have been included in the paper_latex_files/ directory. 

To recompile the paper run the LaTeX build recipe: pdflatex $\to$ bibtex $\to$ pdflatex'x2. All sources are included in the main.bib Bibtex file.
